# @squirrel-forge/ui-core
A collection of interface, classes, functions and abstracts made for the browser and babel compatible.

## Installation

```
npm i @squirrel-forge/ui-core
```

## Usage

```
import { Class, function } from '@squirrel-forge/ui-core';
```
